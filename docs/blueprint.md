# **App Name**: Hanoi Food Explorer

## Core Features:

- Flavor-Based Search: Allows users to filter dishes based on a variety of flavors such as sour, spicy, sweet, savory, and more complex combinations.
- Ingredient-Based Search: Enables users to find dishes using ingredient filters and an autocomplete search bar that will have category selections.
- Database Integration: Store food items, flavors, ingredients, and restaurant details in Firestore database.
- Dish Information Display: Show food image, shot description, cooking recipe url and restaurant address and GG Maps url
- Search Autocomplete Tool: Use an AI tool to give autocomplete suggestions for dish name

## Style Guidelines:

- Primary color: Soft orange (#FFB347) for a warm and inviting feel, reflecting the comfort of home-cooked meals.
- Background color: Light beige (#F5F5DC), a desaturated version of the primary color to complement the orange tones while maintaining a neutral backdrop.
- Accent color: Muted red (#E64A19), chosen for its vibrancy and contrast with the analogous soft orange, drawing attention to key elements.
- Body font: 'PT Sans', a modern and humanist san-serif font for clear and accessible readability
- Headline font: 'Playfair', a transitional serif with a elegant feel.
- Use simple, clear icons to represent different flavor profiles and ingredient categories.
- Use a card-based layout to display food items, making it easy for users to browse and filter options.
